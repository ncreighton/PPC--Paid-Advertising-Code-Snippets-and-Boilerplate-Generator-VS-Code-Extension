## Installation

### From VS Code Marketplace
1. Open VS Code
2. Press `Ctrl+P` (or `Cmd+P` on Mac)
3. Type `ext install digitalforge.8d0a90e0`
4. Press Enter

### From VSIX file
1. Download the `.vsix` file
2. In VS Code: `Extensions > ··· > Install from VSIX`
3. Select the downloaded file

## Usage

Press `Ctrl+Shift+P` and search for `PPC & Paid Advertising Code Snippets and Boilerplate Generator VS Code Extension` to see all available commands.

**Keyboard shortcut:** `Ctrl+Shift+Alt+A` (or `Cmd+Shift+Alt+A` on Mac)

## Settings

| Setting | Default | Description |
|---------|---------|-------------|
| `8d0a90e0.enabled` | `true` | Enable/disable the extension |
| `8d0a90e0.autoRun` | `false` | Auto-run on file open |
| `8d0a90e0.logLevel` | `"info"` | Log verbosity |
